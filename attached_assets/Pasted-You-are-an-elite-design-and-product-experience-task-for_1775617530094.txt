# 3. VISUAL HIERARCHY SYSTEM

Enforce:

* Clear primary action (CTA)
* Strong typography scale
* Proper spacing (breathing room)
* Section grouping

* Clutter
* Redundant elements
* Weak visual contrast

---

# 4. COMPONENT REDESIGN

Upgrade ALL components:

* Inputs → cleaner, responsive, focus states
* Buttons → clear priority (primary/secondary)
* Cards → structured, informative, interactive
* Tables → readable, sortable, modern

Add:

* Hover states
* Focus states
* Disabled states

---

# 5. INTERACTION DESIGN (CRITICAL)

Every action must give feedback.

Define:

### Click:

* Subtle scale (0.97–0.99)
* Instant response

### Hover:

* Soft elevation or glow

### Input:

* Smooth focus highlight
* Validation feedback

# 6. ELITE ANIMATION SYSTEM

Animations must be:

* Smooth (60fps)
* Fast (150–300ms)
* Purposeful (not decorative)

## B. COMPONENT ANIMATIONS

* Cards → fade + rise
* Modals → scale + fade

---

## C. MICRO-INTERACTIONS

* Button press feedback
* Drag-and-drop smooth movement
* Success animations (subtle checkmark)

---

## D. DATA ANIMATIONS

* Numbers count up (not jump)
* Pricing updates animate smoothly
* Progress bars animate
